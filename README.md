# Django Web "My Movie List"

## Requirements

django = 2.2.7

psycopg2 = 2.8.4

django-crispy-forms = 1.8.0

## How to start

1. Install libraties:
 - pip istall django
 - pip install psycopg2
 - pip install django-crispy-forms

2. Run migration
 - python manage.py makemigrations
 - python manage.py migrate

3. Run app
 - python manage.py runserver
